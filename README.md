
# kine

Minecraft utility mod.

(Client side only, cheats optional)

## Current Features

- Display player speed (total and ground).

## Planned

- Render mob health.
- Settings menu.
- Fall prevention (don't walk off ledge, not anti damage cheat).
- Elytra autopilot.
- Projectile aimbot.

## Long Term

- Pathfinding.
- Combat.
